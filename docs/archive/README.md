early-implementation phase (when the codebase was ~60% complete with several
open bugs). They are kept for reference only and are **no longer maintained**.

For the current, authoritative documentation see:
- **`PROJECT_SUMMARY.md`** (repo root) — architecture & current status
- **`DOCUMENTATION_INDEX.md`** (repo root) — index of all live docs
